# 横板过关游戏 - Phaser 3

这是一个逐步迭代开发的 2D 横板平台过关游戏，类似马里奥，核心爽点是「跑、跳、踩怪、吃金币、过关」。

项目目录：`/root/.openclaw/游戏/`

## 技术栈

- **游戏引擎**：Phaser 3
- **构建工具**：Vite
- **语言**：JavaScript / ES Modules
- **物理系统**：Arcade Physics
- **运行环境**：现代浏览器

## 当前功能

截至第 17 步，游戏已经可以完成多关卡横板过关循环，并拥有明确的游戏状态机：

- 游戏状态机
  - `title`：标题 / 开始画面
  - `playing`：正式游玩
  - `gameOver`：失败画面
  - `levelClear`：过关画面
  - 不同状态显示不同画面
  - 非 `playing` 状态下停止玩家移动和碰撞处理
- 标题画面
  - 显示游戏标题
  - 显示当前关卡名称
  - 显示开始提示
  - 支持 `Enter` / `Space` / 点击屏幕开始
- 失败画面
  - 显示 `Game Over`
  - 显示失败原因
  - 显示当前分数
  - 支持 `Enter` / `Space` / `R` 重新开始
- 过关画面
  - 显示过关成功
  - 显示当前关卡名称
  - 显示分数
  - 第一关过关后自动进入下一关
  - 最后一关过关后回到关卡选择
  - 支持 `Enter` / `Space` 进入下一关 / 回到关卡选择
  - 支持 `R` 重玩当前关
- 关卡选择界面
  - 可选择第一关 / 第二关
  - 点击进入对应关卡
- 横板卷轴场景
  - 固定 `960 x 540` 游戏画布
  - 摄像机跟随玩家
  - 关卡宽度大于屏幕宽度
- 玩家控制
  - `← / →` 或 `A / D` 移动
  - `↑ / W / Space` 跳跃
  - `R` 重新开始当前关卡
- 平台与地形
  - 起点平台
  - 地面平台
  - 悬空平台
  - 坑洞与掉落重生
- 金币与分数
  - 金币收集
  - `+10` 分数提示
  - 分数 UI 实时更新
- 敌人与踩怪
  - 巡逻怪兽
  - 从上方踩怪可击败敌人
  - 侧面碰撞会触发失败画面
- 终点与过关
  - 到达终点后显示过关画面
  - 显示最终分数
  - 可进入下一关或回到关卡选择
- 反馈效果
  - 吃金币粒子
  - 踩怪粒子
  - 受伤屏幕闪红
  - 过关闪光与震屏
- 简单音效
  - 吃金币音效
  - 踩怪音效
  - 受伤音效
  - 过关音效
  - 使用 Web Audio API 合成，不依赖外部音频文件

## 运行方式

### 安装依赖

```bash
cd /root/.openclaw/游戏
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后打开终端输出的本地地址，例如：

```text
http://localhost:5173/
```

### 构建生产版本

```bash
npm run build
```

构建输出目录：

```text
dist/
```

### 本地预览生产构建

```bash
npm run preview
```

## 操作说明
