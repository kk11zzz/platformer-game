## 当前操作问题

玩家不知道怎么开始关卡。

当前游戏启动后默认进入 `GameScene` 的 `title` 状态。  
正确开始方式：

- 按 `Enter`
- 按 `Space`
- 或点击屏幕

开始后进入 `playing` 状态，玩家才能移动、跳跃、吃金币和踩怪。

## 操作说明

| 操作 | 按键 |
|------|------|
| 开始游戏 | `Enter` / `Space` / 鼠标点击 |
| 向左移动 | `←` / `A` |
| 向右移动 | `→` / `D` |
| 跳跃 | `↑` / `W` / `Space` |
| 重开当前关卡 | `R` |
| 失败后重新开始 | `Enter` / `Space` / `R` |
| 过关后进入下一关 / 回到选择 | `Enter` / `Space` |
| 过关后重玩当前关 | `R` |
