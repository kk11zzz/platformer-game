# 横板过关游戏 - 开始关卡问题记录

时间：2026-06-17 23:25 GMT+8

## 问题

用户反馈：

```text
Enter
Space
鼠标点击三种方式都无法开始
```

## 排查结果

`GameScene` 里同时使用了两套开始监听逻辑：

1. `update()` 里用 `Phaser.Input.Keyboard.JustDown` 监听 `Enter` / `Space`。
2. `showTitle()` 里又用 `this.input.keyboard.once('keydown-ENTER')` / `once('keydown-SPACE')` 监听。

问题原因：

- `once` 监听只触发一次。
- 如果事件在标题状态切换前被浏览器/Phaser 消费，后续点击和按键可能无法再次触发。
- `Space` 还同时承担“开始”和“跳跃”，在标题状态与 playing 状态切换时容易误判。

## 修复

将标题画面的开始监听改为：

```js
this.input.keyboard.on('keydown-ENTER', this.startPlaying, this);
this.input.keyboard.on('keydown-SPACE', this.startPlaying, this);
```

并保留鼠标点击：

```js
this.input.once('pointerdown', () => {
  this.audio.unlock();
  this.startPlaying();
});
```

同时在 `startPlaying()` 里增加保护：

```js
if (this.gameState !== 'title') return;
```

避免重复开始。

## 用户操作

部署完成后，开始游戏方式：

- 按 `Enter`
- 按 `Space`
- 鼠标点击画面

开始后：

- `← →` / `A D` 移动
- `↑` / `W` / `Space` 跳跃
- `R` 重开
