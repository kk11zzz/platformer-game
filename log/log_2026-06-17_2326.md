# 横板过关游戏 - 关卡选择体验优化日志

时间：2026-06-17 23:26 GMT+8

## 问题

用户反馈：

```text
Enter
Space
鼠标点击三种方式都无法开始
```

实际原因是：

1. 游戏启动后先进入 `LevelSelectScene` 关卡选择页。
2. 关卡选择页只支持鼠标点击关卡按钮。
3. 点击关卡后才进入 `GameScene`。
4. `GameScene` 的 `title` 状态才支持 `Enter` / `Space` / 鼠标点击开始。

因此用户在关卡选择页按 `Enter` / `Space` 没反应，是正常现象。

## 修复

优化 `LevelSelectScene`：

- 关卡选择页现在支持：
  - 鼠标点击关卡按钮
  - 键盘按 `1` 选择第 1 关
  - 键盘按 `2` 选择第 2 关
  - 按 `Enter` / `Space` 开始当前选择关卡
- 关卡按钮增加 hover 高亮。
- 增加当前选择提示。
- 点击/按 Enter 进入关卡后，`GameScene` 自动延迟 180ms 进入 `playing` 状态。
- 这样用户不需要经历“选择关卡后再按一次开始”。

## 当前操作方式

打开：

```text
https://kk11zzz.github.io/platformer-game/
```

然后：

- 鼠标点击「第 1 关」
- 或按 `1`，再按 `Enter` / `Space`
- 或按 `2`，再按 `Enter` / `Space`

进入游戏后：

- `← →` / `A D` 移动
- `↑` / `W` / `Space` 跳跃
- `R` 重开
